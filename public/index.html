AOE2 Museum

A small collection of Age of Empires II tools and archives. No accounts, no tracking; most tools run entirely in your browser.

McMinimap Render isometric minimap PNGs from recorded games and scenarios. Runs the AOE2-McMinimap Python renderer in your browser via Pyodide. Live Scenarios A community archive of Age of Empires II custom scenarios — browse, filter and download thousands of maps, and contribute your own saves. Live
More soon Additional AoE2 tools will land here as the museum grows. Planned

Source on GitHub — the museum is a plain Cloudflare Worker that serves static assets; per-app Python runs client-side.
